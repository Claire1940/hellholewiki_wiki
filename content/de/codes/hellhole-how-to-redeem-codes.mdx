Um deine Belohnungen erfolgreich einzufordern, musst du den korrekten Prozess im Spiel-Client befolgen. Die Benutzeroberfläche kann für neue Spieler manchmal verwirrend sein, insbesondere wenn Updates das Layout des Einstellungsmenüs ändern. <Steps>
  <Step num="1" title="Einstellungen öffnen">

  </Step>
  <Step num="2" title="Reiter 'Konto' finden">
    Wähle im Einstellungsmenü den Reiter "Konto" (Account) aus der linken Navigationsleiste.
  </Step>
  <Step num="3" title="Einlösemenü öffnen">
    Klicke auf die Schaltfläche mit der Aufschrift "Code einlösen" (Redeem Code), um das Texteingabefeld zu öffnen.
  </Step>
  <Step num="4" title="Eingeben und Bestätigen">
    Gib den aktiven Code genau wie geschrieben ein oder füge ihn ein, und klicke dann auf die Schaltfläche "Bestätigen" (Confirm), um die Belohnung zu verarbeiten.

</Steps>

<Section icon={Shield} title="Fehlerbehebung und Fehlervermeidung">

Es kommt häufig vor, dass beim Versuch, Promo-Gegenstände einzufordern, Fehler auftreten. Die meisten Probleme resultieren aus kleinen Syntaxfehlern oder Serververzögerungen bei Events mit hohem Datenverkehr. Zu wissen, wie man diese Probleme behebt, spart Zeit und verhindert Frustration.

Wenn ein Fehler auftritt, spamme die Bestätigungsschaltfläche nicht wiederholt, da dies dein Konto vorübergehend für weitere Einlöseversuche sperren kann.

| Fehlermeldung | Wahrscheinliche Ursache | Empfohlene Maßnahme |
|---|---|---|
| Code abgelaufen | Der Code hat sein Enddatum erreicht | Überprüfe aktive Listen auf neuere Codes |
| Ungültiger Code | Tippfehler oder falsche Groß-/Kleinschreibung | Kopiere den Code und füge ihn erneut ein |
| Bereits eingelöst | Der Code wurde bereits für dieses Konto beansprucht | Überprüfe das Inventar auf Belohnungsgegenstände |
| Server-Zeitüberschreitung | Hoher Datenverkehr oder Verbindungsabbruch | Warte fünf Minuten und versuche es erneut |

<Callout type="warning" title="Best Practice für Kopieren und Einfügen">
Kopiere Codes immer direkt aus offiziellen Listen, anstatt sie manuell einzutippen, um Verwechslungen bei ähnlichen Zeichen wie O und 0 zu vermeiden.
</Callout>

Regionale Einschränkungen können ebenfalls eine erfolgreiche Einlösung verhindern. Einige Codes sind exklusiv für bestimmte Server wie Nordamerika oder Europa. Wenn du auf einem regionalen Server spielst, stelle sicher, dass der von dir verwendete Code mit deiner Region kompatibel ist.
