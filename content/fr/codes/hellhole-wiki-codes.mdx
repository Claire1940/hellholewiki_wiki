<Section icon={Target} title="Comment Échanger des Codes dans Hellhole">

Le processus d'échange de code est simple, mais il nécessite de naviguer à travers quelques menus. Le système est conçu pour livrer les récompenses directement dans votre boîte aux lettres afin d'éviter le débordement de l'inventaire. Suivez ces étapes pour vous assurer que vos récompenses sont réclamées avec succès.

Avant de commencer, assurez-vous que votre client de jeu est mis à jour vers la dernière version. L'utilisation d'une version obsolète du jeu peut parfois causer des erreurs de communication avec le serveur d'échange, entraînant l'échec des réclamations.

<Steps>
  <Step num="1" title="Accéder au Menu des Paramètres">
    Lancez le jeu et attendez que l'écran principal du lobby se charge. Localisez et appuyez sur l'icône d'engrenage dans le coin supérieur droit de l'écran pour ouvrir le menu des paramètres.
  </Step>
  <Step num="2" title="Naviguer vers l'Onglet Compte">
    Dans le menu des paramètres, regardez la barre de navigation de gauche et sélectionnez l'onglet Compte. Cette section contient votre identifiant utilisateur et les options de liaison de compte.
  </Step>
  <Step num="3" title="Saisir le Code d'Échange">
    Appuyez sur le bouton Code d'Échange pour ouvrir la boîte de dialogue de saisie. Tapez ou collez soigneusement le code dans le champ de texte et appuyez sur le bouton Confirmer.
  </Step>
  <Step num="4" title="Réclamer les Récompenses dans la Boîte aux Lettres">
    Quittez le menu des paramètres et revenez au lobby principal. Ouvrez votre boîte aux lettres en jeu, localisez le courrier système contenant vos récompenses, et appuyez sur Tout Réclamer.
  </Step>
</Steps>

<Callout type="info" title="Conseils d'Échange">
Si les récompenses n'apparaissent pas immédiatement dans votre boîte aux lettres, redémarrez le client du jeu. Pendant les heures de pointe, le décalage du serveur peut retarder la livraison jusqu'à dix minutes.
</Callout>

Si vous rencontrez des problèmes lors du processus d'échange, reportez-vous au tableau de dépannage ci-dessous. La plupart des erreurs sont simples à corriger et ne nécessitent pas de contacter le support client.

| Message d'Erreur | Cause Probable | Solution |
| :--- | :--- | :--- |
| Code Invalide | Faute de frappe ou casse incorrecte | Copiez et collez le code directement |
| Code Expiré | La période de promotion est terminée | Consultez la liste active pour les codes actuels |
| Déjà Échangé | Le code a déjà été utilisé sur ce compte | Vérifiez l'historique de votre boîte aux lettres |
| Erreur Réseau | Connexion Internet instable | Passez à un réseau Wi-Fi stable |
