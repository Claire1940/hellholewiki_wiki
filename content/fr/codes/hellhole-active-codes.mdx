<Section icon={Target} title="Comment activer les codes dans Hellhole">

Le processus d'activation dans Hellhole est directement intégré à l'interface de jeu, mais il peut être légèrement caché pour les nouveaux venus. Pour réclamer avec succès vos récompenses des **codes actifs Hellhole**, suivez cette procédure standardisée. Assurez-vous que votre connexion Internet est stable, car le jeu doit vérifier le code avec le serveur central en temps réel.

<Steps>
  <Step num="1" title="Lancer le jeu et se connecter">
    Ouvrez le client Hellhole et assurez-vous d'être connecté à votre profil de personnage principal. Les récompenses sont liées au personnage actif lors de l'activation.
  </Step>
  <Step num="2" title="Accéder au menu des paramètres">
    Localisez l'icône d'engrenage dans le coin supérieur droit de l'écran. Dans la version mobile, cela peut se trouver dans l'onglet « Plus » de la barre de navigation inférieure.
  </Step>
  <Step num="3" title="Localiser l'onglet Promo">
    Dans le menu des paramètres, recherchez un sous-onglet intitulé « Promotions » ou « Activer le code ». Cela ouvrira un champ de saisie de texte.
  </Step>
  <Step num="4" title="Saisir le code">
    Tapez ou collez le code exactement tel qu'il apparaît dans notre liste. Rappelez-vous que les **codes actifs Hellhole** sont généralement sensibles à la casse.

  <Step num="5" title="Récupérer via la boîte aux lettres">
    Une fois que le message « Succès » apparaît, accédez à votre boîte aux lettres en jeu pour collecter les objets. Ils ne sont pas ajoutés automatiquement à votre inventaire.
  </Step>
</Steps>

<Callout type="warning" title="Erreur d'activation">
Si vous recevez une « Erreur réseau » lors de l'étape 4, ne ressaisissez pas le code immédiatement. Vérifiez d'abord votre boîte aux lettres, car la récompense peut avoir été traitée malgré le bug visuel.
