<Section icon={Shield} title="Resolución de problemas y prevención de errores">

Es común encontrar errores al intentar reclamar artículos promocionales. La mayoría de los problemas se deben a pequeños errores de sintaxis o al retraso del servidor durante eventos de mucho tráfico. Saber cómo solucionar estos problemas ahorra tiempo y evita la frustración.

Si ocurre un error, no pulses repetidamente el botón de confirmar, ya que esto puede bloquear temporalmente tu cuenta para realizar más intentos de canje.

| Mensaje de error | Causa probable | Acción recomendada |
|---|---|---|
| Código expirado | El código ha llegado a su fecha de finalización | Consulta las listas activas para ver códigos más nuevos |
| Código no válido | Error tipográfico o mayúsculas incorrectas | Copia y pega el código de nuevo |
| Ya canjeado | El código ya fue reclamado en esta cuenta | Verifica el inventario para ver los artículos de recompensa |
| Tiempo de espera agotado | Alto tráfico o caída de conexión | Espera cinco minutos e inténtalo de nuevo |
